# Mawrid Backend (Alpha)

Backend API for the Mawrid AI knowledge platform.

## Overview

Mawrid Backend is the core API responsible for managing knowledge bases, documents, and AI processing workflows.

It handles:

- Knowledge base and document management
- File uploads and storage integration
- Dispatching AI ingestion jobs
- Receiving processing updates from the AI service

The backend coordinates with the Mawrid AI service to process documents and store searchable knowledge embeddings.
